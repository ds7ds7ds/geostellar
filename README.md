# GeoStellar

**Precision Geothermal Engineering**

Premium geothermal engineering consultancy bringing Oil & Gas industry standards to the sustainable energy sector.

## Overview

GeoStellar provides technical expertise to contractors, design firms, and engineering companies across the Northeast US. We specialize in:

- Design & Engineering
- Execution & Supervision
- Permits & Licensing
- Thermal Conductivity Testing
- Quality Control
- Directional & Inclination Measurements
- Fiber Temperature Measurements (DTS)
- Flow & Pressure Tests
- System Commissioning

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- React Router
- Lucide Icons

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
